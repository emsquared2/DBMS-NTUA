# H.F.R.I. - DBMS Course Project NTUA

This project was undertaken as part of the DBMS course of the Electrical & Computer Engineering class at the National Technical University of Athens.

### Project Overview:

The primary objective of this course was to familiarize students with Database Management Systems and SQL. Specifically, our project involved the following steps:

- Creation of a hypothetical database tailored for the Hellenic Foundation for Research and Innovation (H.F.R.I.).
- Generation and loading of data into the database.
- Designing and execution of queries using SQL.
- Development of a basic web application to facilitate CRUD operations and provide visual representations of query results.
